🚀 Начало работы с SaaSPro

Добро пожаловать в SaaSPro! Эта документация поможет вам быстро освоиться с платформой.

Регистрация

1. Перейдите на страницу /register

2. Заполните форму: email, имя, пароль

3. Подтвердите email (письмо придёт в течение 5 минут)

Первый вход

После входа в систему вы попадёте на дашборд, где сможете:

  • Создать первый проект
  • Настроить уведомления
  • Подключить API

Создание заявки

Для создания индивидуальной заявки перейдите в раздел Индивидуальная разработка и заполните форму.

// Пример запроса к API
curl -X POST https://api.saaspro.ru/v1/ai/ask \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"message":"Привет!"}'

Поддержка

Если у вас возникли вопросы, обращайтесь в поддержку 24/7: support@saaspro.ru или Telegram @saaspro_support
