# Subscription Clean - Production Version

## Установка

1. Скопируйте .env.example в .env
2. Заполните секреты в .env
3. Запустите go run main.go

## Безопасность

- Все секреты хранятся в .env (игнорируется git)
- Для продакшена используйте переменные окружения
- Ключи шифрования храните отдельно

## Восстановление из бэкапа

Секреты хранятся отдельно в зашифрованном виде.
